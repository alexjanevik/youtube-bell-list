# YouTube Bell List

A Firefox extension to view a list of your subscribed YouTube channels that have notifications enabled (because YouTube amazingly doesn't have a feature for this).

## Usage

Install the extension, navigate to https://www.youtube.com/feed/channels and wait a **long** time for this page to load (depending on how many channels you are subscribed to).
